# Max_LOD_Generator
Creates Level of Detail (LODs) for the selected objects in 3ds Max.
This script will create 3 LOD levels with 50%, 25%, and 10% of the original vertex count.

# To run this script in 3ds Max:
 1. Go to Scripting > Run Script...
 2. Select this file (AutoLod.py).
 3. Make sure you have one or more objects selected in your scene.
